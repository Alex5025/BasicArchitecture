# basic-web

Vue 3 前端應用，獨立於 Maven 管理.

## 技術棧

- **Vue 3** — Composition API
- **Pinia** — 狀態管理
- **Vue Router** — HTML5 History Mode 路由
- **Axios** — HTTP 客戶端
- **Vite** — 開發伺服器與建置工具

## 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器 (port 3000)
npm run dev

# 建置生產版本
npm run build
```
